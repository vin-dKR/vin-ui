# Props
| Prop       | Type                                      | Default                        | Description                                                  |
|------------|-------------------------------------------|--------------------------------|--------------------------------------------------------------|
| `href`     | `string`                                  | `-`                            | The URL to open when the button is clicked                   |
| `text`     | `string`                                  | `"This is a lil cute button"`  | The text displayed on the button                             |
| `variant`  | `"black" \| "white" \| "purple" \| "blue" \| "red"` | `"purple"`                    | The color variant of the button's gradient background        |
| `className`| `string`                                  | `""`                           | Additional CSS classes for custom styling                    |
